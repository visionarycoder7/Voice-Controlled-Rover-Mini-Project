Arduino Radar System 🔍

A simple radar system using Arduino Uno, ultrasonic sensor (HC-SR04), and servo motor. The system scans surroundings and visualizes objects using Processing.

Features:
Real-time object detection
Servo-based 180° scanning
Distance measurement using ultrasonic sensor
Radar visualization using Processing
Expandable with buzzer alerts
🛠️ Components Used
Arduino Uno R3
HC-SR04 Ultrasonic Sensor
SG90 Servo Motor
Jumper Wires
🔌 Circuit Connections
Ultrasonic Sensor
VCC → 5V
GND → GND
TRIG → D7
ECHO → D8
Servo Motor
VCC → 5V
GND → GND
Signal → D9
💻 How to Run
1. Upload Arduino Code
Open radar.ino in Arduino IDE
Select correct COM port
Upload code
2. Run Processing Code
Open radar.pde in Processing
Update COM port
Run the sketch
